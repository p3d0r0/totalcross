Desenvolvimento Mobile Multiplataforma com Java?

Desenvolva uma vez e rode em qualquer dispositivo.

TOTALCROSS GLOBAL MOBILE PLATFORM

TotalCross

TotalCross é formada por uma equipe de alta tecnologia com experiência em desenvolvimento mobile em diversas plataformas móveis. Nosso trabalho começou com PALM OS quando criamos a Plataforma Superwaba, avançando para o BlackBerry, Windows CE, Windows Mobile e muito mais. Agora, com a TotalCross, nosso produto suporta as principais plataformas do mercado, como iOS, WP8, Android, Desktops e Coletores (Handhelds da Honeywell, Motorola entre outros).

Plataforma de desenvolvimento mobile TotalCross

TotalCross é uma plataforma de desenvolvimento mobile projetada para ajudar você a construir apps mobile com a linguagem JAVA (tm), usando todos os benefícios da portabilidade entre dispositivos como iPhone, Android, Windows Phone 8, Windows Desktop e Linux.

VANTAGENS

Confira as vantagens que somente os desenvolvedores TotalCross têm.


INDEPENDENTE DE IDE OU OS

Você é um usuário do Eclipse (tm)? Deseja usar o Windows para criar aplicativos para iOS? Sinta-se livre para usar qualquer IDE ou sistema operacional que você deseja para criar seus aplicativos.

SIMULATOR

Não perca tempo baixando SDKs específicos, você não precisa deles. TotalCross tem tudo que você precisa para o desenvolvimento de apps mobile, incluindo um simulador para testar o seu software em diferentes tamanhos de tela.

PORTABILIDADE REAL

Seu aplicativo terá o mesmo comportamento, facilidade de uso e interface de usuário em todas as plataformas implementadas, mesmo em desktops.

IOS APPS SEM UM MAC, É REAL?

SIM! Com TotalCross você pode desenvolver seus aplicativos iOS sem um Mac e distribuí-lo com Apple’s Enterprise Program ou publicá-lo direto na App Store(tm).

APLICATIVOS MÓVEIS PARA IOS & WINDOWS DESENVOLVIDOS COM JAVA

Não perca tempo aprendendo outra linguagem de programação, IDE ou APIs. Use todo o seu conhecimento JAVA (tm) para criar aplicativos para iOS, Windows Phone e Android.

DATABASE INCORPORADO

TotalCross tem dois bancos de dados incorporados. Ele vem com Litebase e SQLite para fornecer o seu aplicativo a melhor persistência no mercado.
